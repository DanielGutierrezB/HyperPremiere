# HyperPremiere

Plugin CEP para Adobe Premiere Pro 2026 que genera gráficos animados por IA
(motor HyperFrames → ProRes 4444 alpha) timados al audio de una clase y los
coloca sobre marcadores de la secuencia.

Ver el diseño completo en `docs/superpowers/specs/2026-07-03-hyperpremiere-design.md`.

**Estado:** spec en revisión. Construcción con Fable 5 vía Claude CLI.
